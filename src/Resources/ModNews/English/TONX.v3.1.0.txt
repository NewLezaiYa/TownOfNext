## Base
- Base on AmongUs v16.0.5
- Base on TOH v5.1.13

## Warning
- Create/join public lobbies is temporarily unavailable
- This change is due to a server-side update that makes modded public lobbies unavailable for vanilla. Private lobbies are not affected.

## What You Need to Know
- This update involves numerous code modifications and there may be a large number of bugs.
- If possible, please report bugs in time and attach logs.

## Fix
- Fix a bug where roles can not be assigned multiple times
- Fix a bug where errors would be shown when connecting to a lobby
- Fix a bug where everyone dies at the end of a game causing certain special skills like Celebrity's skill to be triggered
- Fix a bug where the haunt menu does not display real roles of players on the modded client side
- Fix a bug where some parts of the main menu UI are not positioned correctly on resolutions other than 16:9
- Fix a bug where the left skill counts of Mayor and Veteran are displayed incorrectly on the modded non-host side
- Fix a bug where the size of players' role name is incorrect

## Changing
- Add role transforming information to the summary text
- Add killer names to the summary text
- Add back kill counts to the summary text (From TOHE v2.3.6)
- Add platform information to the lobby list
- Players die in SoloPVP mode instead of being transported into the black room and will be revived automatically after the countdown

## New Feature
- Assign Mode: Role Drafting (Experimental)

## Role Changing
- Jackal can recruit Sidekick
- Eraser can erase any player, including Impostors which will be transformed into vanilla Impostor and Neutrals and Crewmates which will be transformed into vanilla Crewmate
- According to the setting, Schrödinger Cat may transform into the killer's role after being killed

## New Role
- Neutral: Sidekick
- Neutral: Swapper (Experimental)
